Input symbols

Input symbols (which can be used as key alias) by layer + keys. When press and hold key z to activate the 'symbols' layer:

  • ‚Üê, ‚Üí, ‚Üë, ‚Üì, ‚ê£, ‚èé, ‚å´, ‚å¶: Paste the symbol.
  • ,: Paste ‚Äπ (for "left side" which looks nicer than <).
  • .: Paste ‚Ä∫ (for "right side" which looks nicer than >).
  • 4: Paste ‚á•
  • 5: Paste ‚éã
  • 6: Paste ‚åò
  • 7: Paste ‚å•
  • 8: Paste ‚åÉ
  • 9: Paste ‚áß
  • 0: Paste ‚á™

Use any key for the layer, also consider using:

  • simlayer('z', 'symbols')
  • hyperLayer('z', 'symbols')
  • duoLayer('z', 'x', 'symbols')

Copy and edit the code below in the online editor:

export const rules = () => [
  layer('z', 'symbols').manipulators([
    withMapper(['‚Üê', '‚Üí', '‚Üë', '‚Üì', '‚ê£', '‚èé', '‚å´', '‚å¶'])((k) =>
      map(k).toPaste(k),
    ),

    { ',': toPaste('‚Äπ'), '.': toPaste('‚Ä∫') },

    withMapper({ 4: '‚á•', 5: '‚éã', 6: '‚åò', 7: '‚å•', 8: '‚åÉ', 9: '‚áß', 0: '‚á™' })(
      (k, v) => map(k).toPaste(v),
    ),
  ]),
]
